Testing Backend Connection

Instructions for Windows

Due to browser security, you cannot make HTTP requests from a file:// URL. Here are your options:

Option 1: Use the standalone version

Open production-planner.html - it works without a backend server

Option 2: Run a simple web server

  1. Open Command Prompt
  2. Navigate to: cd C:\Users\HP\MeatProductionPlanner
  3. Run: python -m http.server 8000
  4. Open browser to: http://localhost:8000/production-planner-api.html
